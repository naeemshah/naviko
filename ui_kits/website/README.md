# Naviko Website UI Kit

## Overview
High-fidelity interactive recreation of the Naviko marketing website.
Source: `naviko/Naviko.html`

## Screens
- `index.html` — Full scrollable marketing site (nav → hero → how it works → features → pricing → CTA → footer)

## Components
- `Nav.jsx` — Fixed blurred navigation bar
- `Hero.jsx` — Full hero section with badge, headline, CTAs
- `HowItWorks.jsx` — 3-step process section
- `Features.jsx` — 2×2 feature card grid
- `Pricing.jsx` — Side-by-side pricing cards
- `Footer.jsx` — Footer with links and legal

## Design notes
- Fonts: DM Sans + DM Mono via Google Fonts
- Max-width container: 1120px
- Section padding: 100px
- Scroll fade-in animations on all sections
